Project Overview

The Red Yes Studio Website is a bold, rich, and ‘punchy’ website. The website is focused on clean typography and abstract/ethereal background imagery that conveys the diversity of work of different theatre productions the studio has developed. The layout is very asymmetric as well, allowing the user to flow down the page in a ‘dreamlike’ state.
